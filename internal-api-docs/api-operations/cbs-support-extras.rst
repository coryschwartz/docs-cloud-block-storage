cbs-support
~~~~~~~  
   
As a Rackspace support technician, there are additional features of the CBS API exposed to you that customers will not have access.
In order to take advantage of these features, you must have the correct priviledges.

* Increased insight for troubleshooting CBS problems

* Snapshot Progress

* Volume Host Attirbute

* Volume Tenant Attribute

* iscsi session information
